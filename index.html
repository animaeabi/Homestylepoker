Readme

How to use Poker Nights.

Quick start

  1. Enter host name, pick optional group, and hit Host a game.
  2. Players join on their phones and tap Add buy‑in.
  3. When done, host clicks Settle to start chip entry.
  4. Players submit remaining chips on their own screen.
  5. Host reviews totals and saves settlement.

Groups

  • Groups are optional. Use them for recurring tables and summaries.
  • Click a group name to manage players, rename, or delete.
  • Lock a group with a phrase; opening it requires the phrase.

Player flow

  • Players are remembered per device and game.
  • During live play they see: buy‑ins, total, and Add buy‑in.
  • When settling starts, buy‑ins lock and Submit chips appears.
  • Settled games show a summary card and lock changes.

Join as player (Smart Join)

  • Use Join as player on the landing page.
  • Enter your name to auto‑find active games in your group.
  • If multiple games are live, you’ll pick one.
  • No active group game? Join by code.

History & stats

  • History shows past games grouped by group.
  • Quarterly Summary includes settled games only and shows who‑pays‑who.
  • Stats ranks players by net profit with a skill score.

Data & privacy

  • This is a Supabase‑backed app. Anyone with the join link can access that game.
  • Groups can be locked with a phrase, but there is no full login system yet.
  • Host is permanent for a game (stored locally on the host device).

Planned / optional upgrades

  • PWA install (add to home screen).
  • CSV export and receipts.
  • Host/admin PIN and stricter access control.
  • Per‑player notes and settlement history.
  • Private accounts with Supabase Auth (optional).
